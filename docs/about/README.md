# 介绍

A UI Components Library

## 安装

```bash
$ npm install imax-ui --save
```

## 使用

### 全局使用

```js
import Vue from 'vue';
import imaxUi from 'imax-ui';

Vue.use(imaxUi);
```

### 局部使用
```js
import Vue from 'vue';
import Button from 'imax-ui/dist/button';

Vue.use(Button);

// or

Vue.component(Button.name, Button);
```

## 浏览器支持
Modern browsers and Internet Explorer 10+.
